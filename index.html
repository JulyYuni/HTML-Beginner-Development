Quem sou

Olá! Meu nome é Júlia Camilo, nasci em 2003 e atualmente curso Ciência da Computação na Universidade Federal Fluminense (UFF).

Também faço Iniciação Científica na Escola Nacional de Saúde Pública Sergio Arouca (ENSP/Fiocruz), onde participo de pesquisas relacionadas ao financiamento do Sistema Único de Saúde por meio das emendas parlamentares.

Antes da graduação, concluí o Ensino Médio Técnico em Produção Audiovisual na FAETEC Adolpho Bloch.

Sou apaixonada por tecnologia, programação, pesquisa científica, desenvolvimento de jogos, audiovisual e patinação.

Meus projetos

Show do Milhão

Versão digital do jogo "Show do Milhão" sem GUI

Ano do desenvolvimento: 2019

Descrição

O projeto consiste na recriação desse jogo de perguntas e respostas, simulando a dinâmica de desafios, pontuação e progressão de fases. Esse projeto foi desenvolvido como conclusão do curso de Programação em Python no Lipe(UFRJ) com objetivo de aplicar na pática os conceitos de lógica de programação aprendidos ao longo do curso. Este foi meu primeiro projeto de computação.


Metodologia

O jogo foi desenvolvido utilizando Python, com aplicação de estruturas condicionais, listas e funções. O projeto foi criado sem interface gráfica, utilizando o terminal como ambiente de interação.

Clarividência

Curta-metragem produzido durante o curso técnico de Produção Audiovisual

Ano de desenvolvimento: 2021

Descrição

"Clarividência" é o curta-metragem que desenvolvi junto ao meu amigo Algrin como projeto final do curso de Produção Audiovisual na Adolpho Bloch.


Metodologia

O desenvolvimento passou pelas etapas de pré-produção, como criação de roteiro, decupagem, preparação dos sets de filmagem; produção, com a gravação das cenas, captação de imagens e direção de cenas; e pós-produção com a edição do vídeo e tratamento do áudio.

Pong

Recriação do jogo Pong

Ano de desenvolvimento: 2024

Descrição

O projeto consiste na criação de uma versão desse jogo de arcade, com controle de nave, inimigos e sistema de pontuação. O objetivo foi desenvolver habilidades iniciais em desenvolvimento de jogos 2d.


Metodologia

O desenvolvimento foi realizado em Python utilizando a biblioteca PPlay, trabalhando com movimentação de objetos, detecção de colisões, controle de estados do jogo e organização de outras funcionalidades.

SpaceInvaders

Recriação do jogo Space Invaders utilizando Python e PPlay

Ano de desenvolvimento: 2024

Descrição

O projeto consiste na criação de uma versão desse jogo de arcade, com controle de nave, inimigos e sistema de pontuação. O objetivo foi aprimorar as habilidades em desenvolvimento de jogos 2d.

Jogo autoral de gerenciamento de restaurante

Ano de desenvolvimento: 2024

Descrição

Panda Café é um jogo de gerenciamento onde o jogador administra pedidos, clientes e recursos de um restaurante. O projeto busca simular desafios de organização e tomada de decisão enquanto o jogador tenta melhorar o desempenho do estabelecimento.


Metodologia

O projeto foi desenvolvido utilizando programação orientada a objetos em Python, com as bibliotecas PPlay e Pygame.
